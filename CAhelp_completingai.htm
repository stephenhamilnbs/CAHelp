Editing an Architects/Contract Administrators Instruction

Each instruction comes with one default Item ready to edit. To add additional Items to the Architects/Contract Administrators Instruction, click on the Add item button in the toolbar.

Once the Item is added, the user can enter a number/letter, enter the title and add some details.

A drop-down box is also present; clicking on the drop-down box will display a list of Item type options to choose from - select the option that is relevant to the Item. For details on how to add additional Item types to the options displayed in the drop down list, see Instruction Types.

The user can insert approximate values in the omit or add columns within the value text boxes.

Hiding/Showing cost information

The Architects/Contract Administrators Instruction has a drop down on the right hand side where the user has the choice to show or hide cost information on the form.

To hide the costs on the Instruction, select 'Hide costs on printed form'. Please Note: this information will still appear on the form displayed in the Contract Administrator Editor for the users reference. The user can later select the 'Show costs on printed form' option to print out forms with the costing information included.

Editing the Distribution List

The Form Distribution list may also need editing, which can be accessed by clicking on the Distribution List tab. The user can select/deselect team members for inclusion on the form Transmittal Sheet by clicking on the checkbox in the send column. For more information on how to edit this, see Form Distribution List.

Each form in the active job has an associated Form Distribution list. Each of these are populated from the Global Distribution lists. For more information on this, see Global Distribution List. Please Note: Changes made in the Global Distribution lists will be reflected in the Form Distribution lists of any subsequently added forms.

Deleting an Item from a form

It is possible to delete Items from an Instruction that have previously been added. Select the Item that the user wants to delete, and then click on the Delete item button in the toolbar. If an item is not selected in the editor the delete button will be disabled. Please Note: Instruction Items cannot be removed from the form if the form has been issued.
