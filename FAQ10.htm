What is the Draft Number on an Instruction and why does it print on the Issued Form?

A useful feature in NBS Contract Administrator is that it is possible to have more than one Draft Instruction in the same job.

It is common working practice to have more than one Instruction in Draft at any one time. This enables the user, for example, to send numerous Draft Instructions to other members of the project team or the contractor for pricing.

The Draft Number is used as a reference number between the user and other members of the project team. When the Form is issued, it will be given an Instruction number that is sequential for the job.

The Draft Number is retained on the Issued Instruction to support cross-referencing between the Draft and Instruction Numbers for the audit trail.
