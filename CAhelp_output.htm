Output Location

The Output Location is a directory where all of the issued forms will be created in a PDF format.

Each Job has their own output folder, with a given name of whatever is entered in the Job reference field in the Job Details tab.

Each folder is generally created at the same place where the main Output folder is situated, which is usually created alongside the datastore. For more information on where to locate the main Output folder, see Default Output Location.

To change the Output location for the Job, click on the browse button:

Browse to an appropriate location and click OK.
