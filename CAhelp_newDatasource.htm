Creating a data store

There are 2 scenarios when the user will be prompted to create a new data store:

    • The software is licenced for the first time.
    • The original data store has been moved or deleted.

In either case, the user will be presented with the Data store setup wizard.

  1. Select the radio buttion for 'Create a new data store' and click Next.

  2. They will then be prompted to select a data store location - this can be either on the local machine for Standalone licence holders, or on a network location to share with other network licence users.

    Click on the browse button to select a location and then click on the Next button.

  3. The next step is the Issuer setup. At this stage the user is required to fill out the details of their organisation:

    Once the details have been completed, click the Finish button. This will create the data store.
