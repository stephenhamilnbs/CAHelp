Completing the Issuer Details

When the user creates a new data store in NBS Contract Administrator, the Issuer Details need to be completed. It is important that this information is correct as it will be displayed on all of the forms, both in draft and issued.

Please Note: The Form Address lines will auto-fill with information input from the Postal Address, however it is editable.

It is possible to change this information at a later date if required. To do so select Tools > Edit Issuer Details from the menu.
