Instruction types

The Instruction types dialog can be accessed from the top toolbar by selecting Tools > Instruction Types.

To add a new instruction type in the current instruction types list enter the name of your instruction in the field and select Add Type.

To remove an instruction type from the list select the name of your instruction and click Delete Type.
