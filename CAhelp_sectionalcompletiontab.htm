Job in Progress - Sectional Completion tab

Please Note: This tab will only be populated with editable fields if details of sections have previously been entered in the Sectional Completion editor.

This tab will list all the sections in the active Job with the following options in each section:

  • Has practical completion been achieved? – If practical completion has been achieved on the selected section, select the Yes option. Enter the date achieved by selecting a date from the calendar drop down.
  • Has the completion date been amended? - If the completion date for the section has been amended, select the Yes option. Enter the amended date for completion by selecting a date from the calendar drop down.
