Tools: Compact and Repair

Sometimes the data store may fail to open when launching NBS Contract Administrator. The external Compact and Repair tool allows the compaction of selected databases - this helps reduce their overall size on the disk drive.
This tool is external to the main software and can be opened by browsing to the shortcut in the Start Menu - select Start > All Programs > NBS > NBS Tools > NBS Contract Administrator Tools. Alternatively simply search for NBS Contract Administrator Tools.
A window will pop up with 4 tabs - select the Compact and Repair tab.

The tool will select the last data store the user was working on, however it is possible to still browse to another database using the Browse button (displayed as '...').

Ticking the checkbox will overwrite the existing database with the new compacted file - keeping it unticked will simply back-up the old database and create a compacted copy.
Click on the Compact and Repair button to begin the process.
